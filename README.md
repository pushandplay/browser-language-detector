## Download

[Latest build][download-url]

## Installation

In a browser:
```html
<script src="BrowserLanguageDetector.js"></script>
```

Using npm:
```shell
$ npm i --save browser-language-detector
```

## Examples

So... it works like this:

```javascript
BrowserLanguageDetector.lang; // en or your browser language
```
